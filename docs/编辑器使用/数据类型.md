# 数据类型

在 Maingraph for MC 中，节点之间通过不同类型的数据进行交互。你可以通过端口的**颜色**快速识别数据的类型。

---

## 类型概念

### 1. 执行流 (EXEC)
- **白色箭头**代表执行流。
- 逻辑总是沿着执行连线从左向右运行。
- 如果一个节点的执行输入端口没有连线，该节点将永远不会被触发（事件节点除外，它们是执行的源头）。

### 2. 数据流
- **彩色圆点**代表数据端口。
- 数据端口用于在节点之间传递具体的信息（如数字或文本）。
- **注意**: 你不能直接将一个类型的输出连接到不同类型的输入（除非使用“类型转换”节点）。

### 3. 类型转换 (Cast)
如果你需要将一个类型转换为另一个类型（例如将数字转换为字符串以便打印），请使用 **逻辑 -> 数学与转换 -> 类型转换** 节点。
