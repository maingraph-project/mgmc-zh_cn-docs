# 蓝图管理器

蓝图管理器是进入 Maingraph for MC 逻辑创作的核心入口。通过该界面，你可以管理所有的蓝图文件，并将其绑定到游戏中的具体对象上。

## 唤起界面

在游戏中，确保你处于 **创造模式** 且 **权限大于2级**（若是多人模式），然后按下快捷键：

- **`Ctrl + M`**：打开蓝图资源管理器。
![alt text](/assets/editor/image.png)

## 核心功能

### 1. 蓝图列表与浏览
界面中心显示了当前世界（单机）或服务器中已存在的所有蓝图文件。
- **选中**：左键点击列表中的条目。
![alt text](/assets/editor/1.gif)
- **打开编辑器**：**双击**条目或点击底部的“打开”按钮，进入蓝图编辑器。
![alt text](/assets/editor/2.gif)
- **刷新**：如果手动在文件夹中添加了文件，点击“刷新”按钮同步列表。
![alt text](/assets/editor/3.gif)

### 2. 创建新蓝图
在顶部的输入框中输入蓝图名称，点击“创建”即可立即生成一个空的蓝图并自动进入编辑器。
![alt text](/assets/editor/4.gif)

### 3. 文件管理 (右键菜单)
在列表中的任意蓝图条目上点击 **右键**，可弹出管理菜单：
![alt text](/assets/editor/5.gif)
- **重命名**：修改蓝图名称
![alt text](/assets/editor/6.gif)
- **删除**：永久删除该蓝图文件。此操作不可撤销，请谨慎操作。
![alt text](/assets/editor/7.gif)

## 蓝图绑定 (Mapping)

点击界面右上角的 **“映射”** 按钮，可以进入蓝图绑定管理页面。
![alt text](/assets/editor/8.gif)

## 多人模式支持

Maingraph for MC 完美支持多人服务器：
- 当你在服务器上打开资源管理器时，列表会自动同步服务器端的蓝图文件。
- 所有的保存、重命名和删除操作都会实时同步到服务器后台。
