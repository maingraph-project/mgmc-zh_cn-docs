# 变量语法

本页说明字符串输入中的**变量插值**规则。

## 适用范围
仅对**字符串输入端口**生效（例如：消息、命令、URL、字符串节点等）。  
数值、布尔、列表等非字符串端口不会触发解析。

## 语法
- `$bp('var')`：读取**蓝图变量** `var`，并自动转为字符串
- `$g('var')`：读取**全局变量** `var`，并自动转为字符串

示例：
- 输入：`Hello $bp('name')`
- 输出：`Hello 张三`（假设 `name = 张三`）

## 注意事项
- 变量名仅支持**单引号**：`$bp('var')`、`$g('var')`  
- 变量不存在时将被替换为空字符串
