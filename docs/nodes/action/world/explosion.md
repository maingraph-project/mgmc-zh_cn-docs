# 爆炸 (Explosion)

**爆炸** 节点允许在世界中的指定坐标产生一次爆炸效果。

## 节点概览
- **分类**: 动作 > 世界动作
- **内部ID**：`mgmc:explosion`
- ![alt text](images/image-3.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (Exec) | 执行流 | 触发该节点的运行。 |
| **半径** (Radius) | 浮点数 (Float) | 爆炸的半径大小。默认为 `3.0`。如果输入小于或等于 0，则强制设为 `3.0`。 |
| **坐标** (XYZ) | 坐标 (XYZ) | 爆炸发生的具体空间位置。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输出** (Exec) | 执行流 | 节点逻辑执行完毕后，触发后续节点的运行。 |

## 行为说明
1. **爆炸强度**：爆炸的破坏力由半径参数决定。
2. **破坏类型**：该节点产生的爆炸类型为 `TNT`，意味着它会破坏方块、伤害实体并产生火花（取决于服务器设置）。
