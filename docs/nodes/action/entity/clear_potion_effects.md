# 清除所有药水效果 (Clear Potion Effects)

**清除所有药水效果** 节点用于立即移除指定实体身上的所有药水效果（正面和负面效果都会被清除）。

## 节点概览
- **分类**: 动作 > 实体动作
- **内部ID**：`mgmc:clear_potion_effects`
- ![alt text](images/image-8.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (Exec) | 执行流 | 触发该节点的运行。 |
| **目标实体** (Entity) | 实体 (Entity) | 要清除效果的生物。如果未连接，则默认尝试作用于触发蓝图的生物。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输出** (Exec) | 执行流 | 节点逻辑执行完毕后，触发后续节点的运行。 |

## 行为说明
1. **全部移除**：该节点会调用 `entity.removeAllEffects()`，一次性移除所有激活的状态效果（如中毒、虚弱、力量、速度等）。
2. **生物限制**：该节点仅对 `LivingEntity`（生物）生效。
3. **实体回退**：如果“目标实体”未提供，将作用于蓝图的触发实体（Trigger Entity）。
