# 替换字符串 (Replace String)

在输入的字符串中查找指定的子字符串，并将其全部替换为新的内容。

## 节点概览
- **分类**: 变量 > 字符串
- **内部ID**：`mgmc:string_replace`
- ![alt text](images/image-5.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (Input) | 字符串 (String) | 原始字符串。 |
| **查找** (Old) | 字符串 (String) | 需要被替换的子字符串。 |
| **替换为** (New) | 字符串 (String) | 替换后的新内容。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输出** (Output) | 字符串 (String) | 替换完成后的新字符串。 |

## 行为说明
1. **全部替换**：该节点会将原始字符串中所有匹配“查找”内容的子字符串全部替换。
2. **字面量匹配**：该替换是基于普通文本匹配的，不支持正则表达式。
3. **空值处理**：如果原始字符串、查找内容或替换内容中任何一个为 `null`，节点将直接返回原始输入（不进行任何操作）。
4. **类型转换**：如果输入端口连接了非字符串类型的值，系统会尝试将其转换为字符串后再进行处理。
