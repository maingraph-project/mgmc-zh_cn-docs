# 弧度转角度 (To Degrees)

将弧度转换为角度。

## 节点概览
- **分类**: 逻辑 > 三角函数
- **内部ID**：`mgmc:to_degrees`
- ![alt text](images/image-7.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (Input) | 浮点 (Float) | 弧度值。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **结果** (Result) | 浮点 (Float) | 角度值 (deg)。 |

## 行为说明
1. **主要行为**：计算 x * 180 / π。
