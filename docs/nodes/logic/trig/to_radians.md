# 角度转弧度 (To Radians)

将角度转换为弧度。

## 节点概览
- **分类**: 逻辑 > 三角函数
- **内部ID**：`mgmc:to_radians`
- ![alt text](images/image-8.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (Input) | 浮点 (Float) | 角度值 (deg)。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **结果** (Result) | 浮点 (Float) | 弧度值 (rad)。 |

## 行为说明
1. **主要行为**：计算 x * π / 180。
