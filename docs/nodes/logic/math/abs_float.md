# 绝对值 (abs_float)

返回给定浮点数的绝对值。

## 节点概览
- **分类**: 逻辑 > 数学与转换
- **内部ID**：`mgmc:abs_float`
- ![alt text](images/image-8.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (input) | 浮点数 (Float) | 需要计算绝对值的数值。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输出** (result) | 浮点数 (Float) | 计算结果：`|input|`。 |

## 行为说明
1. **主要行为**：该节点返回输入数值的绝对值。无论输入是正数还是负数，结果始终为非负数。
2. **即时运算**：该节点属于数据节点，不包含执行流（Exec）。每当其输出端口被引用时，都会根据当前的输入值重新计算。
