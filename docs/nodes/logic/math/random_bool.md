# 随机布尔值 (概率) (random_bool)

根据给定的概率随机返回“真”或“假”。

## 节点概览
- **分类**: 逻辑 > 数学与转换
- **内部ID**：`mgmc:random_bool`
- ![alt text](images/image-17.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **概率 (0-1)** (chance) | 浮点数 (Float) | 返回“真”的概率。范围应在 `0.0` 到 `1.0` 之间。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **条件** (result) | 布尔值 (Boolean) | 随机结果。如果随机数小于概率值，则返回 `true`。 |

## 行为说明
1. **主要行为**：生成一个 `0.0` 到 `1.0` 之间的随机数，若其小于输入的 **概率** 值，则输出为 `true`。
2. **概率参考**：
   - `0.0`：始终返回 `false`。
   - `0.5`：50% 几率返回 `true`。
   - `1.0`：始终返回 `true`。
3. **默认值说明**：概率默认设为 `0.5`。
4. **即时运算**：该节点属于数据节点，不包含执行流（Exec）。每当其输出端口被引用时，都会根据概率重新判定。
