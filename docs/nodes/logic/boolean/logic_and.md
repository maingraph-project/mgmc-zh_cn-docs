# 逻辑与 (AND)

当所有输入条件均为真时，输出真。

## 节点概览
- **分类**: 逻辑 > 布尔逻辑
- **内部ID**：`mgmc:logic_and`
- ![alt text](images/image.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **A** (a) | 布尔值 (Boolean) | 第一个逻辑条件。 |
| **B** (b) | 布尔值 (Boolean) | 第二个逻辑条件。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **条件** (condition) | 布尔值 (Boolean) | 只有当 A 和 B 均为 `true` 时，输出 `true`；否则输出 `false`。 |

## 行为说明
1. **逻辑运算**：执行标准的布尔“与”运算（AND）。
2. **默认值**：如果输入未连接，默认值通常为 `true`。
3. **类型转换**：非布尔类型的输入会通过 `TypeConverter.toBoolean` 转换为布尔值。
