# 逻辑或 (OR)

只要有一个输入条件为真，输出即为真。

## 节点概览
- **分类**: 逻辑 > 布尔逻辑
- **内部ID**：`mgmc:logic_or`
- ![alt text](images/image-1.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **A** (a) | 布尔值 (Boolean) | 第一个逻辑条件。 |
| **B** (b) | 布尔值 (Boolean) | 第二个逻辑条件。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **条件** (condition) | 布尔值 (Boolean) | 当 A 或 B 其中至少有一个为 `true` 时，输出 `true`；如果两者均为 `false`，则输出 `false`。 |

## 行为说明
1. **逻辑运算**：执行标准的布尔“或”运算（OR）。
2. **默认值**：如果输入未连接，默认值通常为 `true`。
3. **类型转换**：非布尔类型的输入会通过 `TypeConverter.toBoolean` 转换为布尔值。
