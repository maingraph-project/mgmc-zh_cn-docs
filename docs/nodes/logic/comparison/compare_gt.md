# 大于 (>)

判断第一个数值是否大于第二个数值。

## 节点概览
- **分类**: 逻辑 > 比较
- **内部ID**：`mgmc:compare_gt`
- ![alt text](images/image-2.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **A** (a) | 浮点数 (Float) | 第一个比较数值。 |
| **B** (b) | 浮点数 (Float) | 第二个比较数值。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **条件** (condition) | 布尔值 (Boolean) | 如果 A 大于 B，则输出 `true`；否则输出 `false`。 |

## 行为说明
1. **数值转换**：节点会将输入 A 和 B 均转换为双精度浮点数（double）进行比较。
2. **严格大于**：只有当 A 的数值明确大于 B 时，才返回 `true`（不包含等于的情况）。
