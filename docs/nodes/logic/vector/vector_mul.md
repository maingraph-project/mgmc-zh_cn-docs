# 向量乘法 (标量) (Vector Mul Scalar)

将向量乘以一个标量（数值）。

## 节点概览
- **分类**: 逻辑 > 向量
- **内部ID**：`mgmc:vector_mul`
- ![alt text](images/image-6.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **A** (A) | 坐标 (XYZ) | 输入向量。 |
| **B** (B) | 浮点 (Float) | 乘数（标量）。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **结果** (Result) | 坐标 (XYZ) | A * B 的结果。 |

## 行为说明
1. **主要行为**：将向量的每个分量乘以标量 B。
