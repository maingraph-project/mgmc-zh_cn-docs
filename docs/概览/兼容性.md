# Maingraph for MC 的兼容性

目前已知的与以下模组的兼容性限制：

1. **IMBlocker**：在蓝图编辑器的右键搜索框中，输入框使用的是自定义实现而非原生组件。安装 IMBlocker 后，在右键搜索框中将无法正常使用输入法。  
   - **临时解决方案**：建议在编辑蓝图时卸载 IMBlocker 或将其禁用。  
   - **预计修复版本**：`v0.3.*`。  
2. **Just Enough Characters（拼音搜索）**：蓝图编辑器的搜索功能目前不支持 Just Enough Characters 提供的拼音搜索。  
   - **预计修复计划**：暂无计划。
