# Maingraph for MC 的兼容性

目前已知的与以下模组的兼容性情况：

1. **IMBlocker**：自 v0.2.2 蓝图编辑器使用 Minecraft 原生 EditBox 后，IMBlocker 已兼容。
2. **Just Enough Characters（拼音搜索）**：蓝图编辑器的搜索功能目前不支持 Just Enough Characters 提供的拼音搜索。  
    - **预计修复计划**：暂无计划。
3. **Physics Mod**：F6 物理UI模式下，蓝图编辑器不会配合进行物理。
    - **预计修复计划**：不予修复。
    - "真的会有人开着这东西编辑吗...?"
4. **ImmediatelyFast**：经测试，安装ImmediatelyFast后蓝图编辑器无明显帧率提升。
