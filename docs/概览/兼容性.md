# Maingraph for MC 的兼容性

目前已知的与以下模组的兼容性情况：

1. **IMBlocker**：自 v0.2.2 蓝图编辑器使用 Minecraft 原生 EditBox 后，IMBlocker 已兼容。
2. **Just Enough Characters（拼音搜索）**：自 v0.2.3 起，蓝图编辑器已通过反射机制完美兼容 JECh。
    - **现状**：当检测到安装了 JECh 模组时，搜索功能将自动支持拼音首字母匹配（包括节点名、分类、端口名等）。
3. **Physics Mod**：F6 物理 UI 模式下，蓝图编辑器不会配合进行物理。
    - **预计修复计划**：不予修复。
    - "真的会有人开着这东西编辑吗...?"
4. **ImmediatelyFast**：经测试，安装 ImmediatelyFast 后蓝图编辑器无明显帧率提升。
