# 打开网页 (Open Web Page)

在客户端默认浏览器中打开指定的 URL 链接。

## 节点概览
- **分类**: 客户端 > 系统工具
- **内部ID**：`mgmc:client_open_url`
- ![alt text](images/image.png)

## 端口定义

### 输入 (Inputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输入** (exec_in) | 执行流 (Exec) | 触发该节点的执行。 |
| **URL** (url) | 字符串 (String) | 要打开的完整网页地址（需包含 http:// 或 https://）。 |

### 输出 (Outputs)
| 端口名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **输出** (exec_out) | 执行流 (Exec) | 节点执行完成后触发后续节点。 |

## 行为说明
1. **主要行为**：当节点被触发时，服务端会将 URL 发送至玩家客户端。客户端通过操作系统的默认浏览器打开该链接。
2. **安全提示**：由于涉及外部链接跳转，玩家的系统可能会弹出安全警告或需要手动确认（取决于系统设置）。
3. **空值处理**：如果 URL 端口未连接或字符串为空，节点将不执行任何操作并直接触发输出流。
4. **错误处理**：如果提供的 URL 格式非法（如缺少协议头），将在客户端日志中记录错误，不会导致游戏崩溃。
