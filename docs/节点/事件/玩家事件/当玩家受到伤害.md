# 当玩家受到伤害

- **分类**: 事件 > 玩家事件
- **内部 ID**: `on_player_hurt`

## 节点功能
当玩家受到任何类型的伤害时触发。输出伤害数值、攻击者 UUID（如果存在）以及受伤玩家的 UUID。

![当玩家受到伤害时节点](https://free.picui.cn/free/2026/01/03/6958cba20777f.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 玩家受伤后的逻辑执行出口。 |
| 伤害数值 | `damage_amount` | 浮点数 (FLOAT) | 玩家受到的伤害点数。 |
| 攻击者 UUID | `attacker_uuid` | 唯一标识符 (UUID) | 造成伤害的实体的 UUID（如果是由生物或玩家造成的）。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 受伤玩家的唯一标识符。 |
