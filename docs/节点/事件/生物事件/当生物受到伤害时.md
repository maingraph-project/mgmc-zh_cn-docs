# 当生物受到伤害时

- **分类**: 事件 > 生物事件
- **内部 ID**: `on_entity_hurt`

## 节点功能
当世界中的任何生物（实体）受到伤害时触发。输出伤害数值、被攻击者（受害者）UUID 以及攻击者 UUID（如果存在）。

![当生物受到伤害时节点](https://free.picui.cn/free/2026/01/03/6958c95155cc8.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 生物受伤后的逻辑执行出口。 |
| 伤害数值 | `damage_amount` | 浮点数 (FLOAT) | 生物受到的伤害点数。 |
| 被攻击者 UUID | `victim_uuid` | 唯一标识符 (UUID) | 受伤实体的唯一标识符。 |
| 攻击者 UUID | `attacker_uuid` | 唯一标识符 (UUID) | 造成伤害的实体的 UUID（如果是由其他实体造成的）。 |
