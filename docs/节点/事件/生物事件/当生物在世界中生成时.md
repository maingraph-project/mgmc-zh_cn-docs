# 当生物在世界中生成时

- **分类**: 事件 > 生物事件
- **内部 ID**: `on_entity_spawn`

## 节点功能
当世界中有新的生物（实体）生成时触发。输出生成位置和该实体的 UUID。

![当生物在世界中生成时节点](https://free.picui.cn/free/2026/01/03/6958c9b9bad81.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 生物生成后的逻辑执行出口。 |
| X | `x` | 浮点数 (FLOAT) | 生成位置的 X 坐标。 |
| Y | `y` | 浮点数 (FLOAT) | 生成位置的 Y 坐标。 |
| Z | `z` | 浮点数 (FLOAT) | 生成位置的 Z 坐标。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 生成实体的唯一标识符。 |
