# 当生物死亡时

- **分类**: 事件 > 生物事件
- **内部 ID**: `on_entity_death`

## 节点功能
当世界中的任何生物（实体）死亡时触发。输出死亡位置、被攻击者（死者）UUID 以及攻击者 UUID（如果存在）。

![当生物死亡时节点](https://free.picui.cn/free/2026/01/03/6958c98dabffe.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 生物死亡后的逻辑执行出口。 |
| X | `x` | 浮点数 (FLOAT) | 死亡位置的 X 坐标。 |
| Y | `y` | 浮点数 (FLOAT) | 死亡位置的 Y 坐标。 |
| Z | `z` | 浮点数 (FLOAT) | 死亡位置的 Z 坐标。 |
| 被攻击者 UUID | `victim_uuid` | 唯一标识符 (UUID) | 死亡实体的唯一标识符。 |
| 攻击者 UUID | `attacker_uuid` | 唯一标识符 (UUID) | 造成死亡的实体的 UUID（如果是由其他实体造成的）。 |
