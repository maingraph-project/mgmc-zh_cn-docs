# 合并列表 (List Join)

- **分类**: 变量 -> 列表
- **内部 ID**: `list_join`

## 节点功能
将列表中的所有项目连接成一个长字符串，并在每两个项目之间插入自定义的分隔符。

![合并列表节点](https://free.picui.cn/free/2026/01/03/69584de930a4c.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表 | `list` | 列表 (LIST) | 需要被合并的原始列表。 |
| 分隔符 | `delimiter` | 字符串 (STRING) | 插入项目之间的文本。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 字符串 | `string` | 字符串 (STRING) | 合并后的长文本。 |

## 逻辑规则
1. **空值处理**：如果输入的列表没有任何内容，输出也将是一个空字符串。
2. **连接逻辑**：节点会遍历列表对象中的每一个项目，将其转换为文本后，用你提供的“分隔符”重新组装。
3. **分隔符位置**：自定义的分隔符只会出现在项目与项目之间，不会出现在生成的字符串的最前端或最后端。
