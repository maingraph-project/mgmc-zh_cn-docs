# 列表包含 (List Contains)

- **分类**: 变量 -> 列表
- **内部 ID**: `list_contains`

## 节点功能
检查指定的列表中是否包含某个特定的项目。

![列表包含节点](https://free.picui.cn/free/2026/01/03/69584de8eb3e1.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表 | `list` | 列表 (LIST) | 被检查的列表。 |
| 项目 | `item` | 任意 (ANY) | 要查找的内容（支持任何类型）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 结果 | `result` | 布尔值 (BOOLEAN) | 如果包含则返回 `True`，否则返回 `False`。 |

## 逻辑规则
1. **空列表处理**：如果输入的列表为空，结果将直接返回 `False`。
2. **匹配精度**：该节点使用精确匹配。如果是字符串则区分大小写。
3. **原生支持**：直接在列表对象中查找，不再依赖字符串解析。
