# 从列表删除 (List Remove)

- **分类**: 变量 -> 列表
- **内部 ID**: `list_remove`

## 节点功能
从指定的列表中删除特定索引位置的项目，并返回处理后的新列表。

![从列表删除节点](https://free.picui.cn/free/2026/01/03/69584de8c437b.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表输入 | `list_in` | 列表 (LIST) | 需要处理的原始列表。 |
| 索引 | `index` | 浮点数 (FLOAT) | 要删除的项目索引（从 0 开始）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表输出 | `list_out` | 列表 (LIST) | 删除项目后的新列表。 |

## 逻辑规则
1. **列表格式**：列表在内部以 `|` 符号作为各项目的连接符。
2. **索引边界**：如果输入的索引超出了当前列表的有效范围（小于 0 或大于等于当前总长度），节点将保持原样，直接输出原始列表。
3. **不可变性**：该操作会产生一个新的列表字符串，并不会直接修改原始变量的值。
