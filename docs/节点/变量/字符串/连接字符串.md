# 连接字符串 (String Concat)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string_concat`

## 节点功能
将两个独立的字符串按顺序首尾相接，合并为一个长字符串。

![连接字符串节点](https://free.picui.cn/free/2026/01/03/6958540b979c2.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 字符串 (STRING) | 拼接的前半部分。 |
| B | `b` | 字符串 (STRING) | 拼接的后半部分。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 字符串 (STRING) | 合并后的文本。 |

## 逻辑规则
1. **顺序拼接**：结果始终是 `A` 在前，`B` 在后。
2. **空端口处理**：如果某个输入端口未连接，该部分将被视为空字符串处理，不会对结果产生影响。
