# 字符串 (String)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string`

## 节点功能
定义一个静态的字符串常量。通常用于设置命令、消息内容、名称等。

![字符串-节点样子](https://free.picui.cn/free/2026/01/03/695842c51c430.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 值 | `value` | 字符串 (STRING) | 你想要定义的文本内容。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 字符串 (STRING) | 定义好的文本内容。 |
---
**提示**：你也可以在其他节点的输入框直接填写文本，但使用独立的 **字符串** 节点可以方便多处复用同一个值。
