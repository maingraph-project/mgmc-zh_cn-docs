# 玩家执行命令

该节点允许你以指定玩家的身份在服务器上执行一条原版 Minecraft 命令。

![玩家执行命令-节点图样子](https://free.picui.cn/free/2026/01/02/6957ea63508a0.png)

## 节点信息
- **分类**: 动作/玩家动作
- **内部 ID**: `run_command_as_player`

## 入参
| 端口名称 | 内部 ID     | 类型           | 说明                                       |
| :--- | :-------- | :----------- | :--------------------------------------- |
| 输入   | `exec`    | 执行流 (EXEC)   | 逻辑入口。                                    |
| UUID | `uuid`    | 唯一标识符 (UUID) | 要执行命令的玩家 UUID。如果留空或无效，将尝试以当前触发蓝图的玩家身份执行。 |
| 命令内容 | `command` | 字符串 (STRING) | 要执行的命令（不带前缀 `/`，例如 `say hello`）。         |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 命令执行后的后续逻辑。 |

## 使用示例
### 场景：让玩家强制喊话
如果你想让某个玩家（UUID 已知）在聊天栏说话：
1. **UUID**: 连接到该玩家的 UUID。
2. **命令内容**: `say 我是被蓝图控制的！`
![一张图片，运行mgrun就让玩家强制说话](https://free.picui.cn/free/2026/01/02/6957e9340cfb4.png)
---
**注意**：执行的命令将遵循该玩家在服务器上的权限等级。如果玩家没有执行该命令的权限，命令可能会执行失败。
