# 打印到聊天栏 (Print Chat)

- **分类**: 动作 -> 玩家动作
- **内部 ID**: `print_chat`

## 节点功能
将一段文本消息发送到玩家的聊天栏中。如果蓝图是由玩家触发的，则仅该玩家可见；否则会广播给服务器上的所有人。

![打印到聊天栏-节点样子](https://free.picui.cn/free/2026/01/03/6958475088263.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `exec` | 执行 (EXEC) | 触发该动作。 |
| 消息 | `message` | 字符串 (STRING) | 想要发送的文本内容。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行 (EXEC) | 消息发送后触发，用于连接后续动作。 |

## 逻辑规则
1. **智能发送目标**：
   - 如果当前蓝图上下文包含 `触发者 UUID`（例如由玩家移动、运行命令触发），消息将**仅发送给该玩家**。
   - 如果没有明确的触发者，消息将**广播给服务器内所有玩家**。
2. **文本处理**：输入的消息会被视为普通文本（Literal Component）发送，暂不支持原版 JSON 文本组件格式。

## 使用示例

### Hello World

1. **当运行 MGRUN 时**: 输出连入**打印到聊天栏**的输入
2. **打印到聊天栏**: 将 `消息` 端口改成 `"Hello, World!"`。
