# 爆炸 (Explosion)

- **分类**: 动作 -> 世界动作
- **内部 ID**: `explosion`

## 节点功能
在世界的指定位置制造一次爆炸。

![爆炸-节点样子](https://free.picui.cn/free/2026/01/03/695848c4c1831.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `exec` | 执行 (EXEC) | 触发该动作。 |
| X 坐标 | `x` | 浮点数 (FLOAT) | 爆炸位置的 X 坐标。 |
| Y 坐标 | `y` | 浮点数 (FLOAT) | 爆炸位置的 Y 坐标。 |
| Z 坐标 | `z` | 浮点数 (FLOAT) | 爆炸位置的 Z 坐标。 |
| 半径 | `radius` | 浮点数 (FLOAT) | 爆炸的威力/半径（默认为 3.0，TNT 为 4.0）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行 (EXEC) | 爆炸发生后触发。 |

## 逻辑规则
1. **破坏力**：该节点触发的爆炸效果等同于 TNT，**会破坏周围的方块**并伤害实体。

## 使用示例

### 场景：精准打击
1. **当运行 MGRUN 时**: 输出连入**爆炸**的输入
2. **爆炸**: 将**当运行 MGRUN 时**的坐标连入，并将 `半径` 设为 `5.0`。
3. **结果**: 执行 `/mgrun your-bp-name some` 的玩家坐标会立即发生剧烈爆炸。

![精准打击-示例](https://free.picui.cn/free/2026/01/03/695849b315687.png)

---
**提示**：请谨慎使用该节点，大半径的爆炸可能会导致服务器卡顿或严重的地图损毁。
