# 除法 (/)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `div_float`

## 节点功能
将第一个输入除以第二个输入，并输出结果。

![除法-节点样子](https://free.picui.cn/free/2026/01/03/69582ffd6da1e.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 浮点数 (FLOAT) | 被除数。 |
| B | `b` | 浮点数 (FLOAT) | 除数。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | A / B 的计算结果。 |

## 使用示例

### 场景：计算玩家当前血量的百分比
1. **[[获取实体信息]]**: 获取玩家的 `health` (当前血量) 和 `max_health` (最大血量)。
2. **除法**: 将 `health` 设为 `A`，`max_health` 设为 `B`。
3. **输出**: 得到的结果即为血量百分比（如 0.5 代表 50%）。

![除法-使用示例](https://free.picui.cn/free/2026/01/03/695830e6509bd.png)

---
**提示**：
- 如果输入端口未连接，默认值为 `1.0`。
- **注意**：如果除数 `B` 为 `0`，节点将输出 `0` 以防止系统崩溃。
