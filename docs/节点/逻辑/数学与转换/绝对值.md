# 绝对值

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `abs_float`

## 节点功能
输出输入数字的绝对值（即去除负号后的正值）。

![绝对值-节点样子](https://free.picui.cn/free/2026/01/03/6958327d7329c.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `input` | 浮点数 (FLOAT) | 需要计算绝对值的数字。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | 输入数字的绝对值。 |

## 使用示例

### 场景：计算玩家距离原点的水平距离
1. **[[获取实体信息]]**: 获取玩家的 `X` 坐标。
2. **绝对值**: 将 `X` 输入该节点。
3. **输出**: 得到的结果即为玩家在 X 轴上距离坐标 0 的格数。

![绝对值-使用示例](https://free.picui.cn/free/2026/01/03/695832ae81bd2.png)

---
**提示**：如果输入端口未连接，默认值为 `0.0`。
