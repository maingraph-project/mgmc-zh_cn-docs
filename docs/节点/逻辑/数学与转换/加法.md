# 加法 (+)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `add_float`

## 节点功能
将两个浮点数（数字）相加并输出结果。

![加法-节点样子](https://free.picui.cn/free/2026/01/03/69583174754d2.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 浮点数 (FLOAT) | 第一个加数。 |
| B | `b` | 浮点数 (FLOAT) | 第二个加数。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | A + B 的计算结果。 |

## 使用示例

### 场景：计算玩家当前高度加 10 后的坐标
1. **[[获取实体信息]]**: 获取玩家的 `Y` 坐标。
2. **加法**: 将 `Y` 作为输入 `A`，输入 `B` 设置为 `10`。
3. **输出**: 得到的结果即为玩家头顶 10 格的高度。

![加法-使用示例](https://free.picui.cn/free/2026/01/03/695831737e64b.png)

---
**提示**：如果输入端口未连接，默认值为 `0.0`。
