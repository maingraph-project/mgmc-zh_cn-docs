# 四舍五入

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `round_float`

## 节点功能
将输入的浮点数按照四舍五入的原则转换为最接近的整数。

![四舍五入-节点样子](https://free.picui.cn/free/2026/01/03/695836d7493f6.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `input` | 浮点数 (FLOAT) | 需要舍入的数字。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | 四舍五入后的整数。 |

## 使用示例

### 场景：玩家当前的粗略高度
1. **[获取实体信息](/节点/变量/实体/获取实体信息)**: 获取玩家的 `Y` 坐标（通常带有很多小数）。
2. **四舍五入**: 将 `Y` 输入该节点。

![四舍五入-示例](https://free.picui.cn/free/2026/01/03/695837247788b.png)

---
**提示**：如果输入端口未连接，默认值为 `0.0`。
