# 布尔逻辑节点 (Boolean Logic)

- **分类**: 逻辑 -> 布尔逻辑
- **包含节点**: 
  - 逻辑与 (AND) (`logic_and`)
  - 逻辑或 (OR) (`logic_or`)
  - 逻辑非 (NOT) (`logic_not`)
  - 逻辑异或 (XOR) (`logic_xor`)

## 节点功能
对布尔值进行基本的逻辑运算。

![逻辑与、或、非、异或-节点样子](https://free.picui.cn/free/2026/01/03/6958281a3aa9c.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 布尔值 (BOOLEAN) | 第一个布尔输入。 |
| B | `b` | 布尔值 (BOOLEAN) | 第二个布尔输入（NOT 节点无此项）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 结果 | `result` | 布尔值 (BOOLEAN) | 逻辑运算后的结果。 |

## 逻辑规则
- **AND**: 只有当 A 和 B 都为 True 时，结果才为 True。
- **OR**: 只要 A 或 B 有一个为 True，结果就为 True。
- **NOT**: 将 True 变为 False，将 False 变为 True。
- **XOR**: 当 A 和 B 不同时（一个为 True 一个为 False），结果为 True。

## 使用示例

### 场景：给在线的 OP 发送特别提示
1. **[获取实体信息](/节点/变量/实体/获取实体信息)**: 连接玩家的 UUID。
2. **大于等于 (>=)**: 将 `权限等级` 设为 `A`，常数 `4` 设为 `B`。（[数值比较](/节点/逻辑/比较/数值比较)）
3. **逻辑与 (AND)**: 
   - 将 `大于等于` 的结果连入 `A`。
   - 将 `获取实体信息` 的 `是否在线` 连入 `B`。
4. **[分支](/节点/逻辑/流程控制/分支)**: 将 `逻辑与` 的结果连入 `条件`。
5. **[打印到聊天栏](/节点/动作/玩家动作/打印到聊天栏)**: 如果为 True，提示玩家“欢迎管理员上线！”。
