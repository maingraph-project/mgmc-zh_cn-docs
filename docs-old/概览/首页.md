# 首页

# 本系列文档已过时，等待重构。目前本文档适用于 0.1.* 版本。



欢迎使用 **Maingraph for MC**！这是一个为 Minecraft 设计的可视化蓝图引擎。Maingraph for MC 允许通过可视化蓝图制作自定义逻辑，而无需编写（或者很少编写）任何代码。

> [!TIP]
> **使用提示**: 请确保你处于 **创造模式**，否则无法打开蓝图编辑器。
