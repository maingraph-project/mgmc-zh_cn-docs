# 浮点数 (Float)

- **分类**: 变量 -> 浮点数
- **内部 ID**: `float`

## 节点功能
定义一个数字常量（支持小数）。它是数学计算和坐标处理中最常用的基础节点。

![浮点数-节点样子](https://free.picui.cn/free/2026/01/03/695843503dfdc.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 值 | `value` | 浮点数 (FLOAT) | 你想要定义的数字（如 `1.0`, `100`, `-5.5`）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 浮点数 (FLOAT) | 定义好的数字值。 |

---
**提示**：虽然名为“浮点数”，但你也可以输入整数（如 `10`），系统会自动将其处理为 `10.0`。
