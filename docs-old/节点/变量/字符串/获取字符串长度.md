# 获取字符串长度 (String Length)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string_length`

## 节点功能
计算并返回指定字符串中字符的总个数。

![获取字符串长度节点](https://free.picui.cn/free/2026/01/03/695853bcd1163.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 字符串 | `string` | 字符串 (STRING) | 需要测量长度的文本。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 长度 | `length` | 浮点数 (FLOAT) | 字符串的长度。 |

## 逻辑规则
1. **计费标准**：每个字符（包括空格、标点符号、特殊符号等）都会被计为 1 个长度单位。
2. **返回格式**：输出值为浮点数格式（例如 `3.0`），以便直接连接到数学计算节点。
3. **空文本**：如果输入的字符串为空，长度将返回 `0`。
