# 大小写转换/修剪 (String Case)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string_case`

## 节点功能
根据指定的模式，对字符串进行大小写转换或首尾空格清理。

![大小写转换/修剪节点](https://free.picui.cn/free/2026/01/03/695852b2432dc.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 字符串 | `string` | 字符串 (STRING) | 需要处理的原始文本。 |
| 模式 | `mode` | 字符串 (STRING) | 处理方式：`UPPER` (转大写), `LOWER` (转小写), `TRIM` (去空格)。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 字符串 (STRING) | 处理后的结果文本。 |

## 逻辑规则
1. **模式支持**：
   - `UPPER`：将字符串中的所有英文字母转换为大写。
   - `LOWER`：将字符串中的所有英文字母转换为小写。
   - `TRIM`：删除字符串开头和结尾处的所有空白字符（如空格、换行符）。
2. **模式不敏感**：在输入模式名称时，不区分大小写（例如输入 `upper` 或 `UPPER` 效果相同）。
3. **无效模式**：如果输入的模式不属于上述三者，节点将直接输出原始字符串，不做任何改变。
