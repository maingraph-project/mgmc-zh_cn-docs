# 替换字符串 (String Replace)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string_replace`

## 节点功能
在指定的字符串中，将所有出现的某个旧文本替换为新的文本。

![替换字符串节点](https://free.picui.cn/free/2026/01/03/6958543a54b7e.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 字符串 | `string` | 字符串 (STRING) | 需要处理的原始文本。 |
| 旧文本 | `old` | 字符串 (STRING) | 需要被替换掉的内容。 |
| 新文本 | `new` | 字符串 (STRING) | 准备换上去的内容。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 字符串 (STRING) | 替换完成后的文本。 |

## 逻辑规则
1. **全部替换**：节点会查找并替换文本中**所有**符合条件的匹配项，而不仅仅是第一个。
2. **空旧文本处理**：如果你没有指定“旧文本”，节点将不会进行任何操作，直接输出原始文本。
3. **内容删除**：如果你将“新文本”留空，节点会把所有匹配到的“旧文本”直接删除。
4. **字面量匹配**：该节点不使用正则表达式，所有的特殊符号（如 `.` 或 `*`）都会被视为普通文字。
