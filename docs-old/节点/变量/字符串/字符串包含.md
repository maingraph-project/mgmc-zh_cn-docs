# 字符串包含 (String Contains)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string_contains`

## 节点功能
检查一个字符串中是否包含另一个指定的子字符串。

![字符串包含节点](https://free.picui.cn/free/2026/01/03/6958546541e8e.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 字符串 | `string` | 字符串 (STRING) | 被搜索的主文本。 |
| 子字符串 | `substring` | 字符串 (STRING) | 要查找的内容。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 结果 | `result` | 布尔值 (BOOLEAN) | 如果包含则返回 `True`，否则返回 `False`。 |

## 逻辑规则
1. **区分大小写**：该节点对大小写非常敏感。例如，在 `"Hello"` 中查找 `"hello"` 会返回 `False`。
2. **空值逻辑**：如果输入的“字符串”或“子字符串”中有一个为空，结果将始终返回 `False`。
