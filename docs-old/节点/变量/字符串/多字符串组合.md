# 多字符串组合 (Multi-String Combine)

- **分类**: 变量 -> 字符串
- **内部 ID**: `string_combine`

## 节点功能
将多个输入字符串按顺序组合（连接）成一个字符串。与“字符串拼接 (A+B)”不同，该节点支持动态添加无限数量的输入端口。

![多字符串组合-节点图样子](https://free.picui.cn/free/2026/01/03/695929d24caee.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| (动态端口) | `input_0`, `input_1`... | 字符串 (STRING) | 需要组合的字符串片段。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 字符串 (STRING) | 所有输入片段按顺序连接后的结果。 |

## 逻辑规则
1. 节点会按照端口添加的顺序（input_0, input_1...）依次读取输入值。
2. 将所有读取到的字符串直接首尾相接。
3. 如果某个端口没有连接或没有输入值，则视为空字符串。

## 使用示例

### 场景：构建复杂的消息
1. **多字符串组合**:
   - `input_0`: "玩家 "
   - `input_1`: (连接“获取实体信息”的名称输出)
   - `input_2`: " 使用了 "
   - `input_3`: (连接某个自定义变量)
2. **[打印到聊天栏](/节点/动作/玩家动作/打印到聊天栏)**: 将组合后的 `输出` 连入 `信息`。
