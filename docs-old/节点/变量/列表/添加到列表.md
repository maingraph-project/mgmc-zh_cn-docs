# 添加到列表 (List Add)

- **分类**: 变量 -> 列表
- **内部 ID**: `list_add`

## 节点功能
向现有列表的末尾添加一个新项目。

![添加到列表节点](https://free.picui.cn/free/2026/01/04/6959f97aba4ba.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表输入 | `list_in` | 列表 (LIST) | 原始列表。 |
| 项目 | `item` | 任意 (ANY) | 要添加的内容（支持任何数据类型）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表输出 | `list_out` | 列表 (LIST) | 添加新项目后的完整列表。 |

## 逻辑规则
1. **类型支持**：现在列表支持存储任何数据类型（如数字、布尔值、甚至是另一个列表），而不再局限于文本。
2. **自动初始化**：如果输入的原始列表为空，将自动创建一个新列表并存入该项目。
3. **追加方式**：新项目总是被添加到列表的最末尾。
