# 分割字符串 (String Split)

- **分类**: 变量 -> 列表
- **内部 ID**: `string_split`

## 节点功能
将一个长字符串按照指定的分隔符拆分成多个部分，并转换为蓝图可以识别的列表格式。

![分割字符串节点](https://free.picui.cn/free/2026/01/03/69584de949416.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 字符串 | `string` | 字符串 (STRING) | 需要被拆分的原始文本。 |
| 分隔符 | `delimiter` | 字符串 (STRING) | 用于拆分的标识符（如逗号、空格等）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表 | `list` | 列表 (LIST) | 拆分后的结果列表。 |

## 逻辑规则
1. **列表对象生成**：该节点会将拆分后的每一个部分存入一个标准的列表对象中。
2. **特殊符号支持**：你可以安全地使用 `.`、`?`、`*` 等特殊符号作为分隔符，节点内部会自动对其进行转义处理。
3. **空分隔符**：如果你没有提供分隔符，节点将直接返回原始字符串（包装成列表格式），不会进行任何拆分。
