# 从列表中随机抽取 (Random from List)

- **分类**: 变量 -> 列表
- **内部 ID**: `random_list_item`

## 节点功能
从输入的列表中随机选择一个项并输出。

![从列表中随机抽取-节点图样子](https://free.picui.cn/free/2026/01/03/69592cdb8687c.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表 | `list` | 列表 (LIST) | 抽取的源列表。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 值 | `item` | 任意 (ANY) | 抽取的项。 |
