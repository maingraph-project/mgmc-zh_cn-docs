# 获取列表项 (Get List Item)

- **分类**: 变量 -> 列表
- **内部 ID**: `get_list_item`

## 节点功能
根据指定的索引位置，从列表中提取出对应的项目。

![获取列表项节点](https://free.picui.cn/free/2026/01/03/69584dec1d322.png)

## 入参
|端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表 | `list` | 列表 (LIST) | 被读取的列表。 |
| 索引 | `index` | 浮点数 (FLOAT) | 要获取的项目位置（从 0 开始）。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 项目内容 | `value` | 任意 (ANY) | 获取到的项目内容（保持原始数据类型）。 |

## 逻辑规则
1. **起始位置**：列表索引从 `0` 开始。即第 1 个项目的索引是 0，第 2 个是 1，以此类推。
2. **越界保护**：如果输入的索引超出了列表的实际长度（小于 0 或大于等于总数），节点将返回空字符串。
3. **空列表**：从空列表中获取任何索引都会返回空字符串。
