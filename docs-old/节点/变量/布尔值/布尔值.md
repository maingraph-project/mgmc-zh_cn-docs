# 布尔值 (Boolean)

- **分类**: 变量 -> 布尔值
- **内部 ID**: `boolean`

## 节点功能
定义一个布尔常量（True 或 False）。通常用于控制分支走向或设置逻辑状态。

![布尔值](https://free.picui.cn/free/2026/01/03/6958430c4caa4.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 值 | `value` | 布尔值 (BOOLEAN) | 勾选为 True，取消勾选为 False。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `output` | 布尔值 (BOOLEAN) | 定义好的布尔状态。 |
---
**提示**：在蓝图编辑器中，布尔值通常以勾选框的形式展现，直观易用。
