# 向下取整 (Floor)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `floor_float`

## 节点功能
将输入的浮点数向下取整，即输出小于或等于该数的最大整数。

![向下取整-节点样子](https://free.picui.cn/free/2026/01/03/6958383c7b461.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `input` | 浮点数 (FLOAT) | 需要取整的数字。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | 向下取整后的整数。 |

## 使用示例

### 场景：判断玩家当前的坐标是否为偶数
1. **[获取实体信息](/节点/变量/实体/获取实体信息)**: 获取玩家的 `X` 坐标。
2. **[取模](/节点/逻辑/数学与转换/取模)**: 将 `X` 设为 `A`，输入 `B` 设置为 `2`。
3. **[绝对值](/节点/逻辑/数学与转换/绝对值)**：将取模结果取绝对值。
4. **[向下取整](/节点/逻辑/数学与转换/向下取整)**：将绝对值结果向下取整。
5. **等于**：将向下取整结果与 `0` 进行比较。（[数值比较](/节点/逻辑/比较/数值比较)）
6. **[分支](/节点/逻辑/流程控制/分支)**：如果比较结果为 `true`，则说明坐标是偶数。

![判断偶数](https://free.picui.cn/free/2026/01/03/6958366173bd0.png)

---
**提示**：如果输入端口未连接，默认值为 `0.0`。
