# 随机布尔值 (Random Boolean)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `random_bool`

## 节点功能
根据给定的概率（0.0 到 1.0）返回 `真` 或 `假`。例如概率为 0.5 时，有 50% 的几率返回 `真`。

![随机布尔值-节点图样子](https://free.picui.cn/free/2026/01/03/69592c5d3e3e6.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 概率 (0-1) | `chance` | 浮点数 (FLOAT) | 返回“真”的几率。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 结果 | `result` | 布尔值 (BOOLEAN) | 随机生成的布尔结果。 |
