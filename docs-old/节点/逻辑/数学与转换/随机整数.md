# 随机整数 (Random Integer)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `random_int`

## 节点功能
生成一个在 `[最小值, 最大值]` 范围内的随机整数（包含最小值和最大值本身）。

![随机整数-节点图样子](https://free.picui.cn/free/2026/01/03/69592d1f5a553.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 最小值 | `min` | 浮点数 (FLOAT) | 随机数的下限。 |
| 最大值 | `max` | 浮点数 (FLOAT) | 随机数的上限。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 结果 | `result` | 浮点数 (FLOAT) | 生成的随机值。 |

---
**提示**：如果 `最大值` 小于 `最小值`，节点会自动交换这两个值以确保逻辑正确。系统会先将输入的浮点数取整后再进行计算。
