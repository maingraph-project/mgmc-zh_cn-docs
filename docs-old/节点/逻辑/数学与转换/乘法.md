# 乘法 (*)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `mul_float`

## 节点功能
将两个浮点数相乘并输出结果。

![乘法-节点样子](https://free.picui.cn/free/2026/01/03/69582fc34eec4.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 浮点数 (FLOAT) | 乘数 1。 |
| B | `b` | 浮点数 (FLOAT) | 乘数 2。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | A * B 的计算结果。 |
