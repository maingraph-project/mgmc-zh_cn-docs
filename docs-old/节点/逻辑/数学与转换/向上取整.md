# 向上取整 (Ceil)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `ceil_float`

## 节点功能
将输入的浮点数向上取整，即输出大于或等于该数的最小整数。

![向上取整-节点样子](https://free.picui.cn/free/2026/01/03/695837d10c7ec.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `input` | 浮点数 (FLOAT) | 需要取整的数字。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | 向上取整后的整数。 |
