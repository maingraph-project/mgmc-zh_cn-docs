# 最小值 (Min)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `min_float`

## 节点功能
比较两个输入的浮点数，并输出其中较小的一个。

![最小值-节点样子](https://free.picui.cn/free/2026/01/03/695838da7fddc.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 浮点数 (FLOAT) | 比较值 1。 |
| B | `b` | 浮点数 (FLOAT) | 比较值 2。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | A 和 B 中的较小值。 |

---
**提示**：如果输入端口未连接，默认值为 `0.0`。
