# 减法 (-)

- **分类**: 逻辑 -> 数学与转换
- **内部 ID**: `sub_float`

## 节点功能
将第一个输入减去第二个输入，并输出结果。

![减法-节点样子](https://free.picui.cn/free/2026/01/03/695832125adb0.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| A | `a` | 浮点数 (FLOAT) | 被减数。 |
| B | `b` | 浮点数 (FLOAT) | 减数。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `result` | 浮点数 (FLOAT) | A - B 的计算结果。 |

## 使用示例

### 场景：计算两个玩家的高度差
1. **[获取实体信息](/节点/变量/实体/获取实体信息)**: 分别获取玩家 1 和玩家 2 的 `Y` 坐标。
2. **减法**: 将玩家 1 的 `Y` 设为 `A`，玩家 2 的 `Y` 设为 `B`。
3. **输出**: 得到的结果即为两者的高度差。

![减法-使用示例](https://free.picui.cn/free/2026/01/03/695831e77ac9b.png)

---
**提示**：如果输入端口未连接，默认值为 `0.0`。
