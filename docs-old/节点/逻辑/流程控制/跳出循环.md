# 跳出循环 (Break)

- **分类**: 逻辑 -> 流程控制
- **内部 ID**: `break_loop`

## 节点功能
立即终止当前正在运行的 **有限循环**。它类似于编程语言中的 `break` 语句。

![跳出循环-节点样子](https://free.picui.cn/free/2026/01/03/6958442203389.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `exec` | 执行 (EXEC) | 触发该节点以终止循环。 |

## 逻辑规则
1. 当该节点被触发时，它会向蓝图引擎发送一个“中断请求”。
2. 最近的一层 **有限循环** 在检测到该请求后，会停止执行剩余的轮次。
3. 循环终止后，流程会直接跳转到循环节点的 `已完成 (Completed)` 端口。

---
**提示**：该节点必须在循环的 `循环体 (Loop Body)` 逻辑链条中被触发才有效。此外，你也可以直接将逻辑连回循环节点的 `break` 输入端口来达到同样的效果。
