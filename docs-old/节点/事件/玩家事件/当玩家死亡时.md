# 当玩家死亡时

- **分类**: 事件 > 玩家事件
- **内部 ID**: `on_player_death`

## 节点功能
当玩家死亡时触发。输出死亡位置和该玩家的 UUID。

![当玩家死亡时节点](https://free.picui.cn/free/2026/01/03/6958cbbe714e0.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 玩家死亡后的逻辑执行出口。 |
| X | `x` | 浮点数 (FLOAT) | 死亡位置的 X 坐标。 |
| Y | `y` | 浮点数 (FLOAT) | 死亡位置的 Y 坐标。 |
| Z | `z` | 浮点数 (FLOAT) | 死亡位置的 Z 坐标。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 死亡玩家的唯一标识符。 |
