# 当玩家进入游戏时

- **分类**: 事件 > 玩家事件
- **内部 ID**: `on_player_join`

## 节点功能
当玩家连接并进入服务器时触发。输出玩家的 UUID 和名称。

![当玩家进入游戏时节点](https://free.picui.cn/free/2026/01/03/6958cae0d474d.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 玩家进入游戏后的逻辑执行出口。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 进入游戏的玩家的唯一标识符。 |
| 名称 | `trigger_name` | 字符串 (STRING) | 进入游戏的玩家名称。 |
