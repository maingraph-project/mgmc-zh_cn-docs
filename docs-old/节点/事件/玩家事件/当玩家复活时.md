# 当玩家复活时

- **分类**: 事件 > 玩家事件
- **内部 ID**: `on_player_respawn`

## 节点功能
当玩家在死亡后重新生成（复活）时触发。输出复活后的位置和该玩家的 UUID。

![当玩家复活时节点](https://free.picui.cn/free/2026/01/03/6958cac2159b4.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 玩家复活后的逻辑执行出口。 |
| X | `x` | 浮点数 (FLOAT) | 复活位置的 X 坐标。 |
| Y | `y` | 浮点数 (FLOAT) | 复活位置的 Y 坐标。 |
| Z | `z` | 浮点数 (FLOAT) | 复活位置的 Z 坐标。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 复活玩家的唯一标识符。 |
