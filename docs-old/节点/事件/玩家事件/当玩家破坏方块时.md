# 当玩家破坏方块时

- **分类**: 事件 > 玩家事件
- **内部 ID**: `on_break_block`

## 节点功能
当玩家在世界中破坏一个方块时触发。输出方块的位置、方块 ID 以及执行破坏动作的玩家 UUID。

![当玩家破坏方块时节点](https://free.picui.cn/free/2026/01/03/6958cb252887c.png)

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 方块被破坏后的逻辑执行出口。 |
| X | `x` | 浮点数 (FLOAT) | 被破坏方块的 X 坐标。 |
| Y | `y` | 浮点数 (FLOAT) | 被破坏方块的 Y 坐标。 |
| Z | `z` | 浮点数 (FLOAT) | 被破坏方块的 Z 坐标。 |
| 方块 ID | `block_id` | 字符串 (STRING) | 被破坏方块的注册名（如 `minecraft:dirt`）。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 破坏方块的玩家的唯一标识符。 |
