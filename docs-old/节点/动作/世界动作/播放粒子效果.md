# 播放粒子效果 (Play Effect)

- **分类**: 动作 -> 世界动作
- **内部 ID**: `play_effect`

## 节点功能
在世界的指定位置播放粒子效果（如烟雾、火焰、心形等）。

![播放粒子效果-节点样子](https://free.picui.cn/free/2026/01/03/69584a4aa59a8.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输入 | `exec` | 执行 (EXEC) | 触发该动作。 |
| 粒子名称 | `effect` | 字符串 (STRING) | 粒子的命名空间 ID（如 `minecraft:flame`, `minecraft:heart`）。 |
| X 坐标 | `x` | 浮点数 (FLOAT) | 播放位置的 X 坐标。 |
| Y 坐标 | `y` | 浮点数 (FLOAT) | 播放位置的 Y 坐标。 |
| Z 坐标 | `z` | 浮点数 (FLOAT) | 播放位置的 Z 坐标。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行 (EXEC) | 粒子播放指令发出后触发。 |

## 逻辑规则
1. **粒子查询**：节点会根据填写的 ID 在 Minecraft 注册表中搜索。如果错误，则不会有任何效果。
2. **默认参数**：目前粒子播放的参数是固定的（数量: 20, 扩散范围: 0.5 格, 速度: 0.05）。

## 使用示例

### 场景：玩家移动时脚下冒火
1. **当玩家移动**: 获取玩家当前的坐标 `x, y, z`。
2. **播放粒子效果**: 
   - `粒子名称` 填入 `minecraft:flame`。
   - 将玩家坐标分别连入 `x, y, z` 端口。
